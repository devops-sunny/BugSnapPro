Privacy Policy for BugSnapPro

Effective Date: 31 March 2026

BugSnapPro is a Chrome extension designed for developers, testers, and QA teams to capture browser bug reports through screen recording and technical diagnostics.

1. Information We May Access

  • Active tab information
  • Screen recording of the active browser tab
  • Console logs
  • Network request diagnostics
  • Cookies related to the active website
  • Local storage and session storage data
  • IndexedDB and PWA-related technical metadata
  • Page title, URL, and technical debugging context

2. How This Information Is Used

  • Help users reproduce browser bugs
  • Generate debugging reports
  • Support development, testing, and QA workflows
  • Improve issue diagnosis and troubleshooting

3. When Data Is Accessed

BugSnapPro only accesses relevant data when the user explicitly interacts with the extension and starts a debugging or recording session.

4. Data Storage

Captured data is stored locally in the browser unless the user explicitly chooses to export or submit it.

5. Data Sharing

BugSnapPro does not sell personal data and does not use user data for advertising, profiling, or marketing.

6. Chrome Web Store Compliance

BugSnapPro’s use of information complies with the Chrome Web Store User Data Policy, including the Limited Use requirements.
